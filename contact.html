Frequently Asked Questions

Quick answers to common questions about Shamba Network.

Shamba Network is an ecological data oracle and Decentralized Monitoring, Reporting & Verification (dMRV) network. It provides geospatial and climate data on-chain to power Web3 applications such as parametric insurance, carbon credit markets, green bonds, and other climate finance solutions. Founded in 2021 and headquartered near Nairobi, Kenya, Shamba has already reached over 3,000 farmers across East Africa.

Node operation is coming soon. When launched, you will be able to download the open-source Shamba SDK, run a node on your own hardware, and earn rewards for providing accurate climate data to the network. Join our community channels on Twitter and Medium to be notified when the node program launches.

Shamba currently delivers data on six EVM-compatible chains: Arbitrum, Avalanche, Ethereum, Optimism, Polygon, and the XRPL EVM Sidechain. The oracle uses Chainlink infrastructure for production-tested security and broad DeFi compatibility. The XRPL EVM Sidechain integration enables XRP-native settlement workflows alongside full EVM smart contract support.

Start by visiting the quickstart guide. You can import the ShambaGeoConsumer or ShambaFireConsumer smart contracts from the Shamba smart-contract-kit (available as an npm module). The Contracts Tool on shamba.app can generate boilerplate code with parametric specifications for your use case.
